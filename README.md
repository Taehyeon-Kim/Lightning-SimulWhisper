# VoiceType (Lightning-SimulWhisper)

음성 입력을 위한 개인용 세팅입니다.

## Quick Start

```bash
# 1. 레포 클론
git clone https://github.com/Taehyeon-Kim/Lightning-SimulWhisper.git
cd Lightning-SimulWhisper

# 2. 가상환경 생성 및 의존성 설치
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 3. VoiceType 실행
python voicetype.py
```

Option(⌥) 키를 눌러 음성 인식을 토글합니다. 메뉴바에서 모델과 언어를 변경할 수 있습니다.
